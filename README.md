# 高级计算器安卓应用

## 📱 项目概述
这是一个**现代化、高性能、全功能的科学计算器**安卓应用，采用**Jetpack Compose**构建，**无第三方依赖**，使用原生**Kotlin**实现所有数学功能。

## ✨ 特色功能
1. **基础计算**：四则运算、百分比、括号处理
2. **科学计算**：三角函数、对数、指数、幂运算
3. **高级数学**：矩阵运算、复数计算、统计分析
4. **工程计算**：电阻电容计算、时间常数、功率计算
5. **单位转换**：角度/弧度、进制转换、物理单位转换
6. **实时计算**：即时结果显示，超快响应

## 🎨 设计特点
- **黑白主题**：简洁美观的黑色主色调
- **玻璃质感**：毛玻璃效果增强视觉层次
- **现代化UI**：符合Material Design 3规范
- **用户友好**：清晰的按钮分组，易于操作
- **科技感**：简约线条，几何形状，立体感

## 📊 技术架构
- **语言**：Kotlin
- **UI框架**：Jetpack Compose
- **API版本**：Android API 34+
- **架构模式**：MVVM
- **数学库**：完全原生实现

## 🔧 使用方法
1. **基础模式**：加减乘除、括号运算
2. **科学模式**：三角函数、对数、指数、平方根
3. **高级模式**：矩阵、复数、统计、微积分
4. **工程模式**：电阻电容、时间常数、功率计算
5. **矩阵模式**：行列式计算、逆矩阵、转置
6. **复数模式**：加减乘除、绝对值、共轭

## ⚙️ 安装和编译
### 使用Android Studio
1. 打开Android Studio
2. 导入项目`CalculatorApp`
3. 连接安卓设备或使用模拟器
4. 点击运行按钮

### 命令行编译
```bash
cd CalculatorApp
gradlew build
gradlew assembleRelease
```
